# SQL Healthcare Analytics (No-Show Analysis)

## Overview
This repository contains SQL queries that analyse appointment no-show behaviour using a healthcare-style dataset.

## Dataset
- `data/appointments.csv`

## Queries
- `queries/no_show_analysis.sql`

## Key Analyses
- Overall no-show rate
- No-show rate by SMS reminder
- No-show patterns by chronic conditions
- Age band analysis

## How to Run
1. Import `appointments.csv` into a SQL engine (SQLite / PostgreSQL / MySQL).
2. Create a table named `appointments` matching the CSV columns.
3. Run the queries in `queries/no_show_analysis.sql`.

## Notes
This is an educational analytics project demonstrating SQL skills in a healthcare reporting context.
